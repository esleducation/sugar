## API
-----------------------------

### isColor(value : Mixed) : Boolean
Check if the passed value is a color

- Privacy : **Public**

- Return : **Boolean** : The check result

Name | Type | Description | Status | Default
------------ | ------------ | ------------ | ------------ | ------------
value | **Mixed** | The value to check | required | 


#### Sample
```language-undefined
isColor('red') => true
isColor('#fff') => true
isColor('hello') => false

```

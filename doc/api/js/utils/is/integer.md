## API
-----------------------------

### isInteger(value : Mixed) : Boolean
Check if the passed value is an integer

- Privacy : **Public**

- Return : **Boolean** : The check result

Name | Type | Description | Status | Default
------------ | ------------ | ------------ | ------------ | ------------
value | **Mixed** | The value to check | required | 


#### Sample
```language-undefined
isInteger(10) => true
isInteger('hello') => false

```

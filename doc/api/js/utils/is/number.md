


-----------------------------
## API
-----------------------------

### isNumber(Mixed value) : Boolean
Check if the passed value is a number

- Privacy : **Public**

- Return : **Boolean** : The check result

Name | Type | Description | Status | Default
------------ | ------------ | ------------ | ------------ | ------------
value | **Mixed** | The value to check | required | 


#### Sample
```js
isNumber(12) => true
isNumber(22.3) => true
isNumber('20') => false

```

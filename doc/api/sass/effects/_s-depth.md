### @mixin({Integer} $depth, {Color} $color = rgba(0,0,0,.2))

- Privacy : **Public**



Name | Type | Description | Status | Default
------------ | ------------ | ------------ | ------------ | ------------
$depth | **{Integer}** | The depth wanted (1,2,3,4,etc...) | required | 
$color | **{Color}** | The drop shadow color | optional | rgba(0,0,0,.2)


#### Sample
```scss
.my-cool-element {
		\@include s-depth(2);
}

```
